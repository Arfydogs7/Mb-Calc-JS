Q&A/Troubleshooting

Nothing is showing up.

  1. Check your internet
  2. Are you using some crappy browser no one should use (read: internet explorer) or is your browser outdated?
    (edit how to tell if your browser is outdated- just google "does [insert browser version here] support HTML5?")
  3. Do you have JavaScript enabled? (I promise I do not put ads)
  4. Do you have an adblocker that is blocking said JavaScript?
  5. If neither of the above fix it, contact me.

What are you doing with the files I upload?
Nothing. They're not being stored.

The scan function isn't working.
There may be a few bugs with processing the results given by the OCR that I haven't caught. Additionally, I'm not entirely sure how well the program works with dotsheets that appear different from the ones made in Pyware for University High School. Just use the manual input function.

Why is this website so ugly?
I think it looks great. I'm still working on CSS and web design isn't my thing, okay.
